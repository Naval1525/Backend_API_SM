# 📱 Social Media API

A **learning-focused backend project** that simulates the core functionality of a social media platform using **Node.js**, **TypeScript**, **Express**, and **Prisma**.

This project is built as part of my journey to learn and apply **real-world backend development practices**, including clean architecture, database modeling with Prisma, and modern API design.

---

## 🚀 Features

- 🔐 **User Authentication** (JWT)
- 📝 **Create Posts**
- ❤️ **Like / Unlike Posts**
- 💬 **Comment on Posts**
- ➕ **Follow / Unfollow Users**
- 💬 **Chat Functionality** (Coming Soon)

---

## 🛠 Tech Stack

- **Node.js + Express** – Backend framework
- **TypeScript** – Type-safe development
- **Prisma ORM** – Database access
- **PostgreSQL** – Relational database
- **JWT** – Authentication
- **Zod** *(optional)* – Schema validation

---

## 📘 Note

> This backend is built **purely for learning and practice purposes**, and not intended for production use. It is part of my exploration into backend architecture and clean code practices using modern tools.
